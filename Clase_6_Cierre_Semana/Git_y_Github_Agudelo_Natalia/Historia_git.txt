Git es un sistema de versionado creado por Linus Torvals con la finalidad de obtner mas velocidad, diseño sporte para desarrollo no linela, completamente distribuido y capaz de manejar grandes proyectos.
Actualmente GitHub pertenece a microsoft dado que desde el nombramiento de Satya Nadella como CEO de Microsoft, la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.
